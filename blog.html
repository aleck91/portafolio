Sitio web CSM para un blog con tematica de Kdramas.

  1. Inicio

Decripción:

Ghost es una plataforma de blogs de código abierto escrita en JavaScript y distribuida bajo la licencia MIT, diseñada para simplificar el proceso de publicación en línea para blogueros individuales, así como para publicaciones en línea.

Para este proyecto se necesita instalar ghost y crear una cuenta, es de paga, yo estoy usando la version de prueba.

Para este tipo de proyecto no es necesario ningun tipo de conocimiento en programación, el sistema esta diseñado para la interacción de los componentes por medio de su interfaz grafica.
